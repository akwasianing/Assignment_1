Dataset Name: Student Performance Data

Description: The dataset contains information about students' gender, race/ethnicity, parental level of education, lunch type, test preparation course, and their performance in math, reading, and writing scores.

	•	Varaibles
	•	gender -  is categorised into male and female
	•	race/ethnicity - is represented by different groups (A,B,C,D)
	•	parental level of education -  is categorised into some high school, high school, some college, associate's degree, bachelor's degree, master's degree
	•	lunch type -  categorised into free/reduced and standard
	•	test preparation course -  none indicates a student did not take test preparation course; completed indicates a student took and finished the test preparation course
	•	math scores -  student's score in math (range: 0-100).
	•	reading scores - student's score in reading  (range: 0-100).
	•	writing scores  - student's score in reading  (range: 0-100).
